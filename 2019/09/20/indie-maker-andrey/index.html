辞职后一年做了7个App

Sep 20 2019
扫描二维码

此文是「IndieMaker独立变现」知识星球人物志系列之第9篇
官网：https://ezindie.com

Andrey的独立创业变现经历是很多人的缩影，从有了自己的产品，过程中经历了痛苦的挣扎，不得不为了生计绞尽脑汁，理想是坐在咖啡厅里敲着代码，但现实是拿着“打狗棒”走在半夜的小路上，甚至在电线杆下面处理Bug。在经历了很多的尝试之后，也终于迎来了自己的收入增长，完成了自己的目标，自己也过的非常开心。最重要的是收获了自己的经验（文章最后有写）。

2018年的3月份Andrey辞掉了工作，到2019年的5月份，14个月的时间做了7个应用，完成了月收入$1,000的目标。

经历概要

2018年3月：辞掉工作，设定1000美元的月收入为目标

2018年3月：Sheet2Site一个Web应用，允许从Google表格创建一个功能齐全的网站，无需编程。

2018年4月：Progress Bar OS X是一款macOS菜单栏应用，它可以准确显示年，月，日的百分比。

2018年7月：Make OS X Great Again修复macOS中烦人的提示，例如在连接iPhone时停止询问是否要更新或禁用打开iTunes。

2018年8月：Preview Hunt一个准备提交产品到Product Hunt的Web应用程序。

2018年10月：MacBook Alarm（获得）一个macOS应用程序，如果有人试图使用你的笔记本，它将会报警。最终我的这个应用被收购，赚了一笔钱。

2018年11月：Encrypt My Photos端到端加密照片存储。
2019年1月：我赢得了Product Hunt年度最佳Maker。

2019年3月：Sheet2Site 2.0。全新版本，包含会员，工作板，活动，旅行等模板：地图，表格，搜索，过滤器，HTML片段
2019年5月：实现了目标

为什么需要14个月？

与一次性销售相比，经常性收入通常难以产生。 用户必须不断地订购产品才能覆盖成本支出。

我有7个产品，但只有2个订阅式的：
1、Sheet2Site (订阅)
2、DarkModeList (订阅)
3、Progress Bar OS X
4、Make OSX Great Again
5、Mac Book Alarm
6、Preview Hunt
7、Encrypt My Photos

如果我的目标是产生经常性收入，那么为什么我要构建一堆单次收费的应用？

当我最初设定这个目标时，我只想要一个稳定的1000美元收入，来支付我的账单，然后做我喜欢的事情。我真的不在乎是经常性收入还是一次性收费。 虽然当时订阅收费对我来说看起来更加稳定。

现实情况是，订阅产品通常难以变现，与一次性付款产品相比需要更长的时间。 人们讨厌订阅。

在我于2018年3月辞掉工作并推出我的第一个SaaS产品Sheet2Site后，它每月100美元至200美元的收入。 得到我的第一个付费客户真的很令人兴奋，但这些钱不足以支付我的账单。

我有足够的积蓄，可持续约3个月

我一开始构建解决我遇到过的问题的应用，可以通过任何方式实现变现。 如果我没有，我就会饿死并住在帐篷里。

我住在巴厘岛一个便宜的房间，吃着街头小吃。

我买不起摩托车，所以没有任何形式的交通工具。 我每天走4-5公里。

因为有很多狗的原因，晚上就更加的危险，不得不拿着一根长棍子护身。

有时我晚上在非常奇怪的地方工作，当周围都已经关门的时候：

我甚至有一张电子表格，告诉我还能生活多少天（银行里的钱/当天的费用）。 我想当我有一些钱可以解决温饱的时候，我会回到基于订阅的应用程序。

这比我想象的要加艰难

当我辞职开始自己做的时候，我想象我会坐在咖啡馆里可以敲一年的代码，但我低估了很多东西。

支付

我每一天早上都开始阅读电子邮件，并希望PayPal没有向我发送死亡邮件“抱歉，我们阻止了您的帐户，我们无法告诉您原因。 你的钱将在6个月后可用。

技术问题

有些事情一直没有奏效。 试图解决一些技术问题的痛苦很常见:)。

持续性

很难保持一致并且持续做一年的事情。 特别是那些具有非常明确的收入标准的，收入并没有增长，有时甚至下降。 在取得一些成功之后，更难以继续做下去。复制同样的成功非常难。

完成它

完成所有我开始的工作真的很难。 特别是在最后阶段。 即便是这个帖子:)。 但是在这一年里，我养成了有始有终的习惯。

情绪

一年中有50％我有焦虑，压力，拖延，不知道该怎么办。 感谢朋友阻止我放弃。

9个月后达到生存点

9个月后，收入达到生存点时，我制作4个应用程序，我决定制作新的订阅产品或改进现有产品，然后选择最后一个。

Sheet2Site 2.0

Sheet2Site是我的硬核开发年（2018年3月）推出的第一个应用，是订阅服务。Sheet2Site是一个可以从Google表格创建网站而无需编写代码的工具。在2018年12月，我决定专注于Sheet2Site，以便能够以1,000美元的MRR(月经常性收入)实现我的目标，并且接下来的4个月之后投入全职开发。

首先，我对现有客户进行调查研究，以确定最大的问题。我知道主要的问题是，人们更关注模板的问题，但它不够好。

我加了很多的新模板：

  • Affiliate
  • Job board
  • Table
  • Travel
  • Tweets
  • Facebook posts

我也加了一些新的功能:

  • 过滤和搜索
  • 详情页
  • 地图
  • HTML片段
  • Amazon推广佣金
  • API (通过Google Sheet来做自己的网站)

总收入

14个月里我所有的应用(不包括MackBook Alarm)的合计收入是3.2万美金。

我现在开心吗？

是的:)。 这种感觉是非常好的，看到人们正在使用我做的应用，获得的收入也足够我支付账单。

我学到的东西

1、追随梦想，做一些你热爱的事情。这种内心的驱动可以帮助你度过正在做的一些无聊的事情。最好是简单地处理困难的事情。
2、与相信你的人在一起。我非常幸运地遇到了帮助我的那些很酷的人。我甚至认为这些功劳都是他们的，而不是我的成就。
3、在你开始做事之后，才开始发现所有重要细节，而不是之前。
4、始终完成你开始的东西（完成闭环）。这非常重要。每次从零到最后构建产品时，你都会学到很多东西。养成完成事情的习惯。大多数人放弃了。刻意练习你的“完成肌肉”。
5、每次成功后都不要放松。将这种动力用于下一步，但是刻意花一些时间来真正享受你的成就，不要立即跳到下一步。
6、这是马拉松，而不是冲刺，所以你需要及时的充电！
6、专注于产品，而不是工具（代码或无代码）。用户不关心你是如何完成的。你的产品应该解决他们的问题。
7、不要过多的停留在验证阶段，只需快速构建它，看看会发生什么。但是不要做很多不能用的垃圾。做的东西确实是能工作的！首先与你的朋友分享（如果他们是潜在用户）

你应该辞掉工作吗？

我建议你在工作的时候开始制作产品。 它更安全，压力更小，特别是如果你有贷款，家庭和孩子。 而你可以在产品将产生最低收入后再决定。

现在就开始！

如果我能做到，你肯定可以做到！
停止思考，开始构建！ 我相信你！
